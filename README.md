# LZW compression tool in Scala

Course work for HelsinkiU CS project.

## Running

Make sure you have JDK 8 and [sbt](https://www.scala-sbt.org/) installed.

Run `sbt assembly` to build a fresh JAR, then run `./lzwpack --help` to see
available options.
